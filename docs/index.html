Weka Machine Learning for Processing

A Library by Rong-Hao Liang, Janet Huang and Wesley Hartogs for the Processing programming environment.
Last update, 05/10/2021.

Download

Download Weka Machine Learning for Processing version 0.0.15 in .zip format.

Installation

Unzip and put the extracted Weka4P folder into the libraries folder of your Processing sketches. Reference and examples are included in the Weka4P folder.

Possible Conflicts

Please not to uninstall/remove any weka.jar from your libraries folder or code folder (in projects) when using this library. There are possible conflicts that can occur.

Keywords. Machine Learning, Weka, Data, classification, regression, neural networks

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of Weka Machine Learning for Processing is available at GitHub, and its repository can be browsed here.
